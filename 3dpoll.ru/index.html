Инструкция по установке маяков «3D Pol»

Отбейте пол лазерным уровнем
Снимите клейкую ленту с защитного основания точечного маяка. Приклейте к основанию плотно прижав.
С помощью лазерного уровня (можно пузырьковым) установите уровень
Сделайте отметку маркером, либо изолентой другого цвета
Залейте наливной пол до отметки на точечном маяке
Удалите излишки маяка после высыхания пола с помощью ножа
